# Scope and Priority

- This file defines default instructions for work under this directory.
- If a repository has its own `AGENTS.md` (or equivalent project policy), the repository-local rules take precedence.

# Working language

- Interact with the user in Japanese.
- Unless the user explicitly requests otherwise, keep code artifacts in English.
- "Code artifacts" includes:
  - README and docs
  - code comments
  - commit messages
  - pull request titles/descriptions
  - issue titles/descriptions/comments
  - variable/function/class names
